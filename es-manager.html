How does this thing work?

Our golang code generates external secret yaml files by interacting with the vault api and creates files with keys present which are to be created as secrets in kubernetes

Okay but why do we need this?

Stingless is a cli wrapper written in golang over iovisor's bcc tools. We plan to extend towards other open-source tools such as libbpf and cilium for greater surface of tracing

How can I use stingless in my infrastructure?

You can set up our logging infrastructure with a single docker compose up command. Contact us if you wish to know more. We are more than happy to connect with new enterprises
